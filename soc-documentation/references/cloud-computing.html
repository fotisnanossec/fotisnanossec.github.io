Cloud Computing

Cloud services, security, and infrastructure

Cloud Service Models

IaaS

Infrastructure as a Service

  • • Virtual machines
  • • Storage
  • • Networking
  • • Example: AWS EC2

PaaS

Platform as a Service

  • • Development platform
  • • Runtime environment
  • • Databases
  • • Example: Heroku

SaaS

Software as a Service

  • • Complete applications
  • • Web-based access
  • • No infrastructure management
  • • Example: Office 365

Deployment Models

Public Cloud

  • • Shared infrastructure
  • • Cost-effective
  • • High scalability
  • • Provider managed

Private Cloud

  • • Dedicated infrastructure
  • • Enhanced security
  • • Full control
  • • Higher cost

Hybrid Cloud

  • • Mix of public/private
  • • Flexible deployment
  • • Data locality
  • • Complex management

Cloud vs On-Premises Costs

Cloud Benefits

  • • Pay-as-you-use model
  • • No upfront hardware costs
  • • Reduced operational overhead
  • • Automatic scaling

On-Premises Benefits

  • • Predictable long-term costs
  • • Full control over resources
  • • No data transfer fees
  • • Compliance certainty
