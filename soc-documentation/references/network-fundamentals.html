Essential networking concepts for SOC analysts

OSI 7-Layer Model

L7 - Application: HTTP, HTTPS, FTP, SMTP, DNS
L6 - Presentation: SSL/TLS, encryption
L5 - Session: NetBIOS, RPC
L4 - Transport: TCP, UDP
L3 - Network: IP, ICMP, routing
L2 - Data Link: Ethernet, MAC
L1 - Physical: Cables, signals

TCP vs UDP

TCP - Reliable

  • • Connection-oriented (3-way handshake)
  • • Reliable delivery with ACK
  • • Ordered packets
  • • Flow control
# TCP Ports 80 - HTTP 443 - HTTPS 22 - SSH 21 - FTP 25 - SMTP

UDP - Fast

  • • Connectionless
  • • No reliability guarantees
  • • Low overhead
  • • Good for streaming
# UDP Ports 53 - DNS 67/68 - DHCP 69 - TFTP 161 - SNMP 514 - Syslog

Network Devices

Hub (L1)

  • • Repeats to all ports
  • • Single collision domain
  • • Half-duplex
  • • Obsolete

Switch (L2)

  • • Learns MAC addresses
  • • Per-port collision domains
  • • Full-duplex
  • • Intelligent forwarding

Router (L3)

  • • Routes between networks
  • • Per-interface broadcast domains
  • • Routing decisions
  • • Runs routing protocols

ARP & NAT

ARP

Maps IP addresses to MAC addresses

# ARP Commands arp -a # View ARP cache arp -d IP # Delete entry ping IP # Generate ARP request # ARP Process 1. Check ARP cache 2. Broadcast ARP request 3. Target responds with MAC 4. Update cache, send frame

NAT

Translates private to public IPs

# NAT Types Static: 1-to-1 mapping Dynamic: Pool of public IPs PAT: Port translation # Linux NAT iptables -t nat -L iptables -t nat -A POSTROUTING \ -o eth0 -j MASQUERADE
