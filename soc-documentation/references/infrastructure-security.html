Server Hardening

Core Hardening Steps

  1. 1. Remove unnecessary services
  2. 2. Update and patch system
  3. 3. Configure strong authentication
  4. 4. Set file permissions
  5. 5. Enable logging and monitoring
  6. 6. Configure firewall rules

Implementation

# Windows Hardening sconfig # Server configuration netsh advfirewall set allprofiles state on auditpol /set /subcategory:"Logon" /success:enable # Linux Hardening systemctl disable unnecessary_service ufw enable chkrootkit
