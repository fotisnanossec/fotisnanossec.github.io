# Fotis Nanos, Portfolio

Personal cybersecurity portfolio site, built with static HTML and Tailwind CSS.

Live at: https://fotisnanossec.github.io

## Structure

- `index.html` - landing page (profile, certifications, skills, writeups, projects, contact)
- `blog/` - technical writeups
  - `devhub-mcp-writeup.html`, `macos-forensics-writeup.html`, `ransomware-attribution-writeup.html`, `disk-forensics-writeup.html`, `memory-forensics-writeup.html`, `lateral-movement-writeup.html`, formal incident/assessment report format
  - `boogeyman-*.html`, `wgel-ctf-writeup.html`, `insights.html`, earlier writeups, indexed via `blog/index.html`
- `assets/images/blog/` - images used in writeup cards
- `Fotios.jpg` - profile photo

## Notes

All writeup pages are `noindex, nofollow` and intentionally omit exact flags, artifact values, and challenge-identifying detail. They document methodology and findings in a report format rather than step-by-step walkthroughs.

No resume file is hosted publicly on this site. Available on request.
